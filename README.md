# بوت AHL لتصميم بوسترات إعلانية للسيارات — دليل التشغيل الكامل

بوت تيليجرام مجاني بالكامل يستقبل صورة منتج ويحوّلها تلقائيًا لبوستر إعلاني احترافي
بهوية الشركة (أسود + أحمر + أبيض)، جاهز للنشر على انستغرام وفيسبوك.

---

## 1. الملفات اللي لازم تضيفها بنفسك قبل التشغيل

هذي الملفات ما نقدر ننزّلها تلقائيًا (تحتاج إنترنت وقت التحميل)، لازم ترفعها بنفسك:

### أ) الشعار
ارفع ملف شعار الشركة (خلفية شفافة PNG أفضل خيار) داخل مجلد:
```
assets/logo/logo.png
```

### ب) الخطوط (لدعم العربي بشكل احترافي)
حمّل هذه الخطوط المجانية من Google Fonts وضعها في `assets/fonts/`:
- **Tajawal** (Bold + Regular): https://fonts.google.com/specimen/Tajawal
- **Cairo** (Bold + Regular): https://fonts.google.com/specimen/Cairo
- **Montserrat** (للنصوص الإنجليزية): https://fonts.google.com/specimen/Montserrat

اسم الملفات المطلوب بالضبط:
```
assets/fonts/Tajawal-Bold.ttf
assets/fonts/Tajawal-Regular.ttf
assets/fonts/Montserrat-Bold.ttf
assets/fonts/Montserrat-Regular.ttf
```
(لو ما رفعتها، البوت بيشتغل بخط افتراضي بديل تلقائيًا بدون ما يتوقف.)

---

## 2. ملف .env (بيانات سرّية)

انسخ ملف `.env.example` وسمّه `.env` ثم املأ:

```
BOT_TOKEN=التوكن اللي أخذته من BotFather
ADMIN_ID=آيدي حسابك في تيليجرام
```

**كيف تعرف ADMIN_ID الخاص فيك؟**
ابحث في تيليجرام عن بوت اسمه `@userinfobot` وأرسل له `/start`، بيعطيك رقمك مباشرة.

---

## 3. رفع المشروع على GitHub (من الهاتف) - طريقة مبسّطة

كل ملفات الكود الآن في المجلد الرئيسي مباشرة (بدون مجلدات فرعية)، فترفعها كلها دفعة وحدة:

1. افتح متصفح الهاتف وادخل github.com وسجّل دخول
2. اضغط **+** أعلى الصفحة ثم **New repository**
3. سمّه مثلاً `ahl-poster-bot` واختر **Private** (خاص)
4. اضغط **Create repository**
5. اضغط رابط **"uploading an existing file"**
6. اختر كل هذه الملفات مرة وحدة (كلها موجودة بنفس المجلد، بدون فتح أي مجلد فرعي):
   bot.py, config.py, database.py, poster_flow.py, admin_flow.py,
   poster_generator.py, background_removal.py, text_utils.py,
   requirements.txt, .env.example, README.md
7. اضغط **Commit changes**

**بعدها، الشعار فقط (ملف واحد داخل مجلد assets/logo):**

8. بصفحة المستودع الرئيسية، اضغط **Add file ← Upload files**
9. اختر ملف `logo.png` من مجلد `assets/logo` بجهازك
10. **قبل ما تضغط Commit**، بصفحة الرفع فيه مربع اسمه فيه `logo.png` — اضغط عليه وعدّله يدويًا ليصير `assets/logo/logo.png` (بإضافة المسار قبل الاسم) — هذا يخلي GitHub ينشئ المجلدات تلقائيًا
11. اضغط **Commit changes**

**ملاحظة:** مجلدات fonts و backgrounds و generated فاضية حاليًا، تجاوزها تمامًا — البوت ينشئها لوحده.

**تذكير: لا ترفع ملف `.env` أبدًا (فقط `.env.example`)** لأنه يحتوي التوكن السري.

## 4. النشر المجاني على Railway

**Railway** يوفر استضافة مجانية تشغّل البوت 24 ساعة بدون ما تسكر جهازك.

1. روح إلى: https://railway.app
2. سجّل دخول بـ **Continue with GitHub**
3. اضغط **New Project**
4. اختر **Deploy from GitHub repo**
5. اختر المستودع (repository) اللي رفعته `ahl-poster-bot`
6. Railway بيكتشف تلقائيًا إنه مشروع بايثون
7. روح لتبويب **Variables** وأضف المتغيرات يدويًا (بما إن .env ما رفعناه):
   - `BOT_TOKEN` = التوكن تبعك
   - `ADMIN_ID` = آيدي حسابك
8. روح لتبويب **Settings** → **Deploy** → تأكد أن **Start Command** مكتوب فيه:
   ```
   python bot.py
   ```
9. اضغط **Deploy**

بعد دقيقة أو دقيقتين، افتح بوتك في تيليجرام واكتب `/start` — لازم يرد عليك فورًا.

> ملاحظة: أول مرة يشتغل فيها البوت، مكتبة `rembg` بتنزّل نموذج الذكاء الاصطناعي لإزالة الخلفية تلقائيًا (حجمه صغير، مرة وحدة فقط)، فأول صورة ممكن تاخذ وقت أطول شوي.

**بديل مجاني آخر لو Railway طلب بطاقة بنكية:** استخدم **Render.com** بنفس الخطوات بالضبط (Deploy from GitHub → أضف نفس المتغيرات → Start Command: `python bot.py`، واختر نوع الخدمة **Background Worker** وليس Web Service لأن البوت لا يحتاج منفذ ويب).

## 5. كيف تستخدم البوت (تجربة فعلية)

1. افتح البوت في تيليجرام واضغط **Start**
2. أرسل صورة المنتج
3. أرسل اسم المنتج
4. أرسل السعر (أو اكتب `تخطي`)
5. أرسل المواصفات، كل ميزة بسطر
6. أرسل التوافق (مثال H4) أو `تخطي`
7. أرسل نص العرض الترويجي أو `تخطي`
8. اختر القالب من الأزرار
9. اختر المقاس (مربع / بورتريه / ستوري / فيسبوك)
10. تستلم صورة PNG جاهزة للنشر ✅

---

## 6. لوحة تحكم الأدمن (بدون كود)

بحسابك (اللي حطيت آيدي فيه بـ ADMIN_ID)، أرسل للبوت:
```
/admin
```
راح تقدر تعدّل: اسم الشركة، الشعار النصي، رقمي الهاتف، والعنوان — كل هذا يُحفظ فورًا في قاعدة بيانات ويظهر في كل بوستر جديد.

## 7. هيكلة المشروع

```
ahl_poster_bot/
├── bot.py                     # تشغيل البوت
├── config.py                  # الإعدادات والقوالب والمقاسات
├── database.py                # SQLite لبيانات الشركة القابلة للتعديل
├── requirements.txt           # المكتبات المطلوبة
├── .env.example                # نموذج ملف الأسرار
├── handlers/
│   ├── poster_flow.py         # أسئلة البوت خطوة بخطوة
│   └── admin_flow.py          # لوحة تحكم الأدمن
├── services/
│   ├── background_removal.py  # إزالة الخلفية + تحسين الصورة
│   ├── poster_generator.py    # محرك تصميم البوستر بالكامل
│   └── text_utils.py          # دعم الكتابة العربية RTL
├── assets/
│   ├── logo/                  # ضع شعارك هنا
│   ├── fonts/                 # ضع الخطوط هنا
│   └── backgrounds/           # (اختياري) خلفيات إضافية مستقبلًا
└── generated/                  # الصور الناتجة تُحفظ هنا تلقائيًا
```

## 8. القوالب المتاحة حاليًا

| المفتاح | الاسم |
|---|---|
| `luxury_black` | Template 1 - Luxury Black |
| `red_racing` | Template 2 - Red Racing |
| `minimal_white` | Template 3 - Minimal White |
| `premium_carbon` | Template 4 - Premium Carbon Fiber |

كل قالب له لوحة ألوان مستقلة داخل `config.py` (`TEMPLATES`)، وتقدر تضيف قالب خامس بسهولة بإضافة سطر جديد بنفس الملف بدون لمس باقي الكود.

---

## 9. أخطاء شائعة وحلولها

| المشكلة | الحل |
|---|---|
| "لم يتم العثور على BOT_TOKEN" | تأكد أنك أضفت `BOT_TOKEN` في Railway/Render Variables |
| النص العربي يظهر معكوس أو منفصل الحروف | تأكد من رفع خطوط Tajawal/Cairo في `assets/fonts/` |
| الشعار ما يظهر بالبوستر | تأكد إن اسم ملف الشعار بصيغة png/jpg وموجود داخل `assets/logo/` |
| البوت لا يرد إطلاقًا | تأكد أن الخدمة "Running" وليست "Crashed" في لوحة Railway/Render، وافحص الـ Logs |

---

هذا هو المشروع كاملًا وجاهز للرفع والتشغيل. إذا احتجت تعديل أي جزء (شكل معين، حجم خط، قالب خامس، دعم QR Code) أخبرني بالتحديد وأعدّله لك مباشرة.
